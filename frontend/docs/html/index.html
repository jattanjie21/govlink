National data platform · The Gambia · Nexu Labs

A serious data platform, built for serious work.

GovLink aggregates open data from Gambian ministries, the Bureau of Statistics, and development partners — UNDP, UNICEF, WHO, the World Bank, AfDB — into one citable, queryable platform. The visual reference is closer to The Pudding × Our World in Data × Stripe Docs than to Kaggle. Editorial gravitas, restrained accent, real Gambian context, and tables that read like a serious newspaper would publish them.

Implementation notes

For the engineering team

  • Tokens map directly to Tailwind theme.extend.colors and CSS custom properties — no derivation required.
  • Each component below maps to a shadcn/ui primitive plus a small custom layer.
  • Tabular numerals via font-variant-numeric: tabular-nums on every number — already on the body's font-feature-settings stack.
  • Dark mode is a single data-theme="dark" attribute on <html>, persisted to localStorage.

shadcn primitives in play

  • Button, Tabs, Card, Table, Sheet (mobile filters)
  • Command (⌘K palette)
  • Dialog (download format picker)
  • Tooltip, Toast, Skeleton (LoadingTable matching final shape)
  • Popover (param help text)

What's intentionally not here

  • No purple gradients, no glassmorphism, no parallax — every anti-pattern in the brief was treated as a hard rule.
  • No flag-as-UI. The single nod is the 24×8 footer stripe.
  • No Inter as a display face. Geist body, Fraunces display.
  • No invented stats. Numbers are plausible Gambian figures (LGAs, ministries, partner orgs); GBoS, MoH, and partner-org names are real.
