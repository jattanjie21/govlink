Engineering · build instructions

Implement screens in React + Tailwind + shadcn, without inventing tokens.

A walkthrough of the GovLink design system for the engineering team. Every component on the platform maps to a shadcn/ui primitive; every visual token is expressed as a CSS variable and a Tailwind theme extension. Copy the snippets on this page into your repo and the screens will render correctly without further design input.

Tailwind theme extension

Use this to expose every token as a Tailwind utility — bg-canvas, text-ink-2, border-rule, font-display, etc. Stub colors point at the CSS variables above so dark mode is automatic.

Typography

Three families. Display (Fraunces) for headlines, dataset titles, KPI numbers. Body / UI (Geist) for everything else. Mono (JetBrains Mono) for code, IDs, hashes. Tabular numerics are mandatory wherever a column of numbers appears.
