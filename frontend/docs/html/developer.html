01

Getting started

Five steps from zero to a working request. The free tier covers 100,000 calls per month — most non-commercial integrations never need to upgrade.

01

Create an account

Use the Sign up form. Government and academic emails are auto-verified; commercial accounts go through a 24-hour manual review.

02

Generate an API key

Visit your developer dashboard and click + New API key. Copy the key immediately — it's shown only once. Test keys are prefixed gl_test_; live keys are prefixed gl_live_.

03

Make your first request

Send the key as a Bearer token. Every endpoint accepts pagination via page and per_page. Below: list health facilities in the West Coast Region.

SHELLCopy# List health facilities in West Coast Region
curl https://api.govlink.gm/v1/health/facilities \
  -H "Authorization: Bearer $GOVLINK_KEY" \
  -d "region=west-coast" \
  -d "per_page=50"
04

Read the response

Every response is JSON with a meta envelope and a data array. Pagination, version, and request ID live in meta.

JSON · 200 OK{
  "meta": { "total": 142, "page": 1, "per_page": 50, "version": "v1.4.2", "request_id": "req_8a4c…" },
  "data": [
    {
      "id": "hf-WC-0042",
      "name": "Brikama District Hospital",
      "type": "district_hospital",
      "region": "west-coast",
      "beds": 128,
      "lat": 13.2705, "lon": -16.6486
    }
  ]
}
05

Install the SDK (optional)

SDKs handle auth, pagination, retries, and typing. Available for Python, JavaScript / TypeScript, and R.

Python

pip install govlink

 JavaScript

npm i @govlink/sdk

 R

install.packages("govlink")

03

Errors

The API uses standard HTTP status codes. Every error response includes a meta.request_id you can quote in support.

200
OK

Request succeeded. data contains the result.

201
Created

Returned on successful POST. The created object is in data.

400
Bad Request

Validation failed. errors[] lists the field-level problems.

401
Unauthorized

Missing, malformed, or revoked API key. Generate a fresh key from the dashboard.

429
Too many requests

Rate limit exceeded. Check the Retry-After header.

500
Server error

Something went wrong on our side. The request_id is logged automatically.
