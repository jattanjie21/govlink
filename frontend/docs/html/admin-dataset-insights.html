Add a region-level summary endpoint

67% of queries on /facilities filter by region. A pre-aggregated /facilities/by-region/summary would cut payload by ~92% and serve from cache.

Index (region, lga, ward_code)

Quality

3 nulls in ownership column

Three rows in the WCR sheet have ownership = null. This filter is used in 22% of /facilities queries, so consumers are quietly dropping these rows. Setting a default of "unknown" keeps them visible.
