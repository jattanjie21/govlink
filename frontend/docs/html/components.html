Component library · v1.0

Every component, every state.

The atoms and molecules of GovLink. Each component lists the shadcn/ui primitive it extends and shows the states the design system expects engineering to wire — default, hover, focus, active, disabled, loading. Data-bearing surfaces also have empty and error states.

§ 01

Buttons

Four variants. Primary uses --accent and is the only filled button on most pages. Secondary is the workhorse. Ghost is for low-stakes actions in dense surfaces. Destructive is reserved for irreversible operations.

§ 06

Filter sidebar extends Accordion + Checkbox

Used on Browse. Faceted by category, organization, region, tags, license. Each group is collapsible; counts come from the search backend and update live. Selected filters surface as removable chips above the result grid.

Default
FiltersClear (3)
Category ▾
Region ▾
License ▾

Behaviour

  • Group headers toggle open/closed; state persists in URL params for shareable filtered views.
  • Counts reflect remaining filters applied (e.g. Health = 94 dataset matches given the Region filter is also active).
  • Clear button shows the count of active filters and removes them all in one click.
  • Mobile collapses the sidebar into a Sheet (shadcn) opened by a "Filter (3)" button above the grid.

§ 09

Code snippet extends Tabs + ScrollArea

Tabbed snippet block in four languages — every API page ships with this. Syntax highlighting uses the GovLink palette (purple keys, ochre strings, forest numbers). Tab selection persists per visitor in localStorage.

# Health facilities in West Coast Region, paginated
import govlink

client = govlink.Client(api_key="$GOVLINK_KEY")

facilities = client.health.facilities.list(
    region="west-coast",
    facility_type="district_hospital",
    per_page=50,
)

for hf in facilities.data:
    print(hf.id, hf.name, hf.beds)

§ 12

Loading state extends Skeleton

Skeletons match the final layout. Bars are the height of one line; widths are 60% / 80% / 95% to imply real content underneath. Shimmer runs at linear 1.4 s.

DatasetOrganizationLast updatedAPI calls (30 d)

Use this only when the wait is > 200 ms. For shorter waits, render the empty layout instantly and let real content swap in. Skeletons that flash are noise.

shadcn mapping summary: Button · Input · Select · Tabs · Checkbox · Accordion · Card · Badge · Avatar · Table · Dialog · Sheet · Toast / Sonner · Pagination · Breadcrumb · Command (⌘K) · Skeleton · Alert · ScrollArea.

Components without a shadcn base — DatasetCard, KpiCard, FilterSidebar, SchemaTable, ApiPlayground, OrgBadge, EndpointHeatmap. These are GovLink-specific compositions on top of the primitives above and are documented in handoff.html.
